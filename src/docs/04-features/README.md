# 功能特性概览

STranslate 提供多种核心功能，本模块文档介绍主要功能的实现细节。

## 子模块

- [热键系统](hotkey.md) - 全局热键、软件内热键、特殊热键功能
- [剪贴板监听](clipboard-monitor.md) - 剪贴板监听实现

## 功能列表

| 功能 | 说明 | 相关文档 |
|------|------|---------|
| 全局热键 | 系统级快捷键，即使应用未聚焦也能触发 | [热键系统](hotkey.md) |
| 软件内热键 | 仅在应用聚焦时生效的快捷键 | [热键系统](hotkey.md) |
| 按住触发键 | 按住特定键时临时激活功能 | [热键系统](hotkey.md) |
| 剪贴板监听 | 监听剪贴板变化并自动翻译 | [剪贴板监听](clipboard-monitor.md) |
